# InputStream зұ»

InputStream зұ»жҳҜеӯ—иҠӮиҫ“е…ҘжөҒзҡ„жҠҪиұЎзұ»пјҢжҳҜжүҖжңүеӯ—иҠӮиҫ“е…ҘжөҒзҡ„зҲ¶зұ»пјҢInputStream зұ»е…·жңүеұӮж¬Ўз»“жһ„

еёёз”Ёж–№жі• (е…¶еӯҗзұ»зҡ„еёёз”Ёж–№жі•еӨ§иҮҙдёҖж ·):

иҝ”еӣһеҖј ж–№жі•еҗҚеӯ— (еҸӮж•°) иҜҙжҳҺ

int available() пјҡиҝ”еӣһд»ҺжӯӨиҫ“е…ҘжөҒдёӯеҸҜд»ҘиҜ»еҸ–пјҲжҲ–и·іиҝҮпјүзҡ„еү©дҪҷеӯ—иҠӮж•°зҡ„дј°и®ЎеҖјпјҢиҖҢдёҚдјҡиў«дёӢдёҖж¬Ўи°ғз”ЁжӯӨиҫ“е…ҘжөҒзҡ„ж–№жі•йҳ»еЎһгҖӮ
void close() пјҡе…ій—ӯжӯӨиҫ“е…ҘжөҒе№¶йҮҠж”ҫдёҺжөҒзӣёе…іиҒ”зҡ„д»»дҪ•зі»з»ҹиө„жәҗгҖӮ
void mark(int readlimit) пјҡж Үи®°жӯӨиҫ“е…ҘжөҒдёӯзҡ„еҪ“еүҚдҪҚзҪ®гҖӮ
boolean markSupported() пјҡжөӢиҜ•жӯӨиҫ“е…ҘжөҒжҳҜеҗҰж”ҜжҢҒ mark е’Ң reset ж–№жі•гҖӮ
abstract int read() пјҡд»Һиҫ“е…ҘжөҒиҜ»еҸ–ж•°жҚ®зҡ„дёӢдёҖдёӘеӯ—иҠӮгҖӮ
int read(byte[] b) пјҡд»Һиҫ“е…ҘжөҒдёӯиҜ»еҸ–дёҖдәӣеӯ—иҠӮж•°пјҢе№¶е°Ҷе®ғд»¬еӯҳеӮЁеҲ°зј“еҶІеҷЁйҳөеҲ— b гҖӮ
int read(byte[] b, int off, int len) пјҡд»Һиҫ“е…ҘжөҒиҜ»еҸ–жңҖеӨҡ len дёӘеӯ—иҠӮзҡ„ж•°жҚ®еҲ°еӯ—иҠӮж•°з»„гҖӮ
byte[] readAllBytes() пјҡд»Һиҫ“е…ҘжөҒиҜ»еҸ–жүҖжңүеү©дҪҷеӯ—иҠӮгҖӮ
int readNBytes(byte[] b, int off, int len) пјҡе°ҶжүҖиҜ·жұӮзҡ„еӯ—иҠӮж•°д»Һиҫ“е…ҘжөҒиҜ»е…Ҙз»ҷе®ҡзҡ„еӯ—иҠӮж•°з»„гҖӮ
void reset() пјҡе°ҶжӯӨжөҒйҮҚж–°е®ҡдҪҚеҲ°жңҖеҗҺеңЁжӯӨиҫ“е…ҘжөҒдёҠи°ғз”Ё mark ж–№жі•ж—¶зҡ„дҪҚзҪ®гҖӮ
long skip(long n) пјҡи·іиҝҮе№¶дёўејғжқҘиҮӘжӯӨиҫ“е…ҘжөҒзҡ„ n еӯ—иҠӮзҡ„ж•°жҚ®гҖӮ
long transferTo(OutputStream out) пјҡд»ҺиҜҘиҫ“е…ҘжөҒдёӯиҜ»еҸ–жүҖжңүеӯ—иҠӮпјҢе№¶жҢүиҜ»еҸ–зҡ„йЎәеәҸе°Ҷеӯ—иҠӮеҶҷе…Ҙз»ҷе®ҡзҡ„иҫ“еҮәжөҒгҖӮ

# OutputStream зұ»

OutputStream зұ»пјҢеӯ—иҠӮиҫ“еҮәжөҒзҡ„жҠҪиұЎзұ»пјҢдёӢеӣҫжүҖзӨәеұӮж¬Ўз»“жһ„пјҡ

еёёз”Ёж–№жі•:

.close() е…ій—ӯжӯӨиҫ“еҮәжөҒе№¶йҮҠж”ҫдёҺжӯӨжөҒзӣёе…іиҒ”зҡ„д»»дҪ•зі»з»ҹиө„жәҗгҖӮ

.flush() пјҡеҲ·ж–°жӯӨиҫ“еҮәжөҒе№¶ејәеҲ¶д»»дҪ•зј“еҶІзҡ„иҫ“еҮәеӯ—иҠӮиў«еҶҷеҮәгҖӮ

.write(int b) пјҡе°ҶжҢҮе®ҡзҡ„еӯ—иҠӮеҶҷе…ҘжӯӨиҫ“еҮәжөҒгҖӮ

.write(byte[] b) пјҡе°Ҷ b.length еӯ—иҠӮд»ҺжҢҮе®ҡзҡ„еӯ—иҠӮж•°з»„еҶҷе…ҘжӯӨиҫ“еҮәжөҒгҖӮ

.write(byte[] b, int off, int len) пјҡд»ҺжҢҮе®ҡзҡ„еӯ—иҠӮж•°з»„еҶҷе…Ҙ len еӯ—иҠӮпјҢд»ҺеҒҸз§»йҮҸ off ејҖе§Ӣиҫ“еҮәеҲ°жӯӨиҫ“еҮәжөҒгҖӮ
