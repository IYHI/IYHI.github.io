# JavaBean

# и®ӨиҜҶ Javabean

еңЁ JSP еҶ…еөҢеӨ§йҮҸзҡ„ java д»Јз ҒдјҡйҖ жҲҗз»ҙжҠӨдёҚж–№дҫҝпјҢеӣ жӯӨи§ЈеҶіиҝҷдёӘй—®йўҳзҡ„еҠһжі•е°ұжҳҜе°Ҷ JSP дёӯзҡ„д»Јз Ғз§»жӨҚеҲ° Java зұ»дёӯ

иҝҷдәӣ java зұ»е°ұжҳҜ JavaBean

еңЁ JavaBean дёӯеҸҜд»Ҙе°ҶйҖ»иҫ‘жҺ§еҲ¶гҖҒеҖјгҖҒж•°жҚ®еә“и®ҝй—®е’Ңе…¶д»–еҜ№иұЎиҝӣиЎҢе°ҒиЈ…

е®һйҷ…дёҠ JavaBean е°ұжҳҜдёҖз§Қ Java з»„д»¶жҠҖжңҜ

дҪңз”Ёпјҡеҗ‘з”ЁжҲ·жҸҗдҫӣе®һзҺ°зү№е®ҡйҖ»иҫ‘зҡ„ж–№жі•жҺҘеҸЈ

JavaBean ж”ҜжҢҒдёӨз§Қз»„д»¶пјҡеҸҜи§ҶеҢ–з»„д»¶е’ҢйқһеҸҜи§ҶеҢ–з»„д»¶

еҸҜи§ҶеҢ–з»„д»¶е’ҢйқһеҸҜи§ҶеҢ–з»„д»¶

еҸҜи§ҶеҢ–з»„д»¶пјҡејҖеҸ‘дәәе‘ҳеҸҜд»ҘеңЁиҝҗиЎҢз»“жһңдёӯзңӢеҲ°зҡ„з•Ңйқўж•Ҳжһң

йқһеҸҜи§ҶеҢ–з»„д»¶пјҡдёҖиҲ¬дёҚиғҪиў«и§ӮеҜҹеҲ°пјҢдё»иҰҒз”ЁеңЁжңҚеҠЎеҷЁз«Ҝ

JSP еҸӘж”ҜжҢҒйқһеҸҜи§ҶеҢ–з»„д»¶

# еҶҷ JavaBean

зј–еҶҷ JavaBean иҰҒжіЁж„Ҹзҡ„дәӢйЎ№пјҡ

  • йҖҡиҝҮ getter/setter ж–№жі•жқҘиҜ» / еҶҷеҸҳйҮҸзҡ„еҖјпјҢеҜ№еә”еҸҳйҮҸйҰ–еӯ—жҜҚеҝ…йЎ»еӨ§еҶҷ

  • еұһжҖ§зҡ„еҗҚз§°з”ұ getter е’Ң setter ж–№жі•еҶіе®ҡ

    жӯӨж—¶пјҢзі»з»ҹдёӯе®ҡд№үзҡ„еұһжҖ§еҗҚз§°дёә xiaomingпјҢиҖҢдёҚжҳҜ str

JavaBean е°ұжҳҜдёҖдёӘ Java зұ»пјҢеҸӘжҳҜе®ғдёҘж јжҢүз…§ JavaBean и§„иҢғзј–еҶҷ

# е®ҡд№ү JavaBean

жңүдёӨз§ҚйҖүжӢ©

# зӣҙжҺҘеңЁ JSP дёӯе®һдҫӢеҢ– JavaBean

< % 
   Student student = new Student();
   // дҪҝз”Ёstudent
   % >

# дҪҝз”Ё <jsp:useBean> ж Үзӯҫ

< jsp:useBean id = "idName" class = "" scope = "page|sessionВ·В·В·" >

idпјҡжҢҮе®ҡ JavaBean еҜ№иұЎзҡ„еҗҚз§°

classпјҡжҢҮе®ҡдҪҝз”Ёе“ӘдёӘзұ»жқҘе®һдҫӢеҢ– JavaBean еҜ№иұЎ

scopeпјҡжҢҮе®ҡеҜ№иұЎзҡ„дҪңз”ЁиҢғеӣҙ

дёҠйқўзҡ„зӯүд»·дәҺпјҡ

< jsp:useBean id="student" class = "Student" ></jsp:useBean>

# и®ҫзҪ® JavaBean еұһжҖ§

# зӣҙжҺҘзј–еҶҷ Java д»Јз Ғ

< jsp:useBean id="student" class = "Student" ></jsp:useBean>
< %
   student.setName("ж„§иҜ—");
   % >

# дҪҝз”Ё <jsp:setProperty> ж Үзӯҫ

дҪҝз”Ёж–№жі•пјҡ

< jsp:setProperty  property = "еұһжҖ§еҗҚ" name = "еҜ№иұЎеҗҚпјҲJavaBeanйҮҢзҡ„idпјү" value = "" >

дёҚд»…д»…еҸҜд»Ҙи®ҫзҪ®еҚ•дёҖзҡ„еұһжҖ§

иҝҳеҸҜд»Ҙе°Ҷе®һдҫӢеҢ–еҜ№иұЎдј е…Ҙ

< jsp:setProperty  property = "*" name = "student" />
<!--е®һдҫӢеҢ–зҡ„еҜ№иұЎstudent-->

иЎЁзӨәе°ҶжүҖжңүе’ҢеұһжҖ§еҗҚзӣёеҗҢеҸӮж•°зҡ„еҖјж”ҫе…Ҙ student еҜ№еә”зҡ„еұһжҖ§дёӯ

# иҺ·еҸ– JavaBean еұһжҖ§

# дҪҝз”Ё JSP иЎЁиҫҫејҸжҲ–иҖ… JSP зЁӢеәҸж®ө

	< % @ page language = "java" import = "Student" contenType = "text/html;charset = gb2312" % >
    < jsp:useBean id="student" class = "Student" ></jsp:useBean>
    < jsp:setProperty  property = "stuname" name = "student" value = "rose" >
    < %= student.getStrname() % >

# дҪҝз”Ё jsp:getProperty

< jsp:getProperty  property = "еұһжҖ§еҗҚ" name = "еҜ№иұЎеҗҚпјҲJavaBeanйҮҢзҡ„idпјү" />

дёҠйқўзҡ„зӯүд»·дәҺ

< jsp:getProperty  property = "stuname" name = "student" />

# DAO е’Ң VO

DAOе’ҢVO

DAO пјҡdata access object ж•°жҚ®и®ҝй—®еҜ№иұЎ

  • дё»иҰҒз”ЁжқҘе°ҒиЈ…еҜ№ DB зҡ„и®ҝй—®пјҲCRUD ж“ҚдҪңпјүгҖӮ
  • йҖҡиҝҮжҺҘж”¶ Business еұӮзҡ„ж•°жҚ®пјҢжҠҠ POJO жҢҒд№…еҢ–дёә POгҖӮ

VOпјҡvalue object еҖјеҜ№иұЎ /view object иЎЁзҺ°еұӮеҜ№иұЎ пјҲе№¶дёҚжҳҜж•°жҚ®жЁЎеһӢпјү

  • дё»иҰҒеҜ№еә”йЎөйқўжҳҫзӨәпјҲweb йЎөйқў /swtгҖҒswing з•Ңйқўпјүзҡ„ж•°жҚ®еҜ№иұЎгҖӮ
  • еҸҜд»Ҙе’ҢиЎЁеҜ№еә”пјҢд№ҹеҸҜд»ҘдёҚеҜ№еә”пјҢиҝҷж №жҚ®дёҡеҠЎзҡ„йңҖиҰҒгҖӮ

жіЁж„Ҹ: дёҠйқўзҡ„ Student зұ»еҸҜд»Ҙе……еҪ“ VO

дҪҝз”Ёиҝҷдәӣзҡ„еҘҪеӨ„жҳҜеңЁ JSP дёӯдёҚдјҡеҮәзҺ°д»»дҪ•дёҺ JDBC жңүе…ізҡ„д»Јз ҒпјҢдҪҝеҫ—жӣҙе®№жҳ“з»ҙжҠӨпјҢж•ҲзҺҮжӣҙй«ҳ
